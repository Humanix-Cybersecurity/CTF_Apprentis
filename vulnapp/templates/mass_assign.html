API de mise à jour de profil

Cette API REST permet de modifier son profil utilisateur.

Endpoint

POST /api/profile/update
Content-Type: application/json

{
  "id": 1,
  "email": "nouveau@mail.lcl",
  "note": "Ma nouvelle note"
}

Champs documentés

ChampTypeDescription
idintID de l'utilisateur à modifier
emailstringNouvelle adresse e-mail
notestringNote personnelle

Exemple avec curl

curl -X POST http://localhost:8080/api/profile/update \
  -H "Content-Type: application/json" \
  -d '{"id": 1, "email": "test@hmnx.lcl"}'
